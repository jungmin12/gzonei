회원탈퇴신청

회원탈퇴는 지존에서의 모든 이용권한 해지를 의미합니다.
반드시 아래 유의사항을 확인 후, 회원 탈퇴 신청을 해주시기 바랍니다.

회원탈퇴 시, 이런점은 유의하세요 !
  • 회원탈퇴 시,
    회원님의 가입정보와 결제 내역 정보 등이
    모두 삭제됩니다.
  • 사업자회원의 경우,
    신청서류 또한 모두 삭제 됩니다.
  • 나의 관심정보에 담으셨던
    개발정보, 경매, 공매 물건 등이 삭제 됩니다.
    (재가입 후에도 내역확인 불가능)
  • 회원이 직접 등록한
    ‘직매물’정보도 삭제됩니다.
  • 단, 상담게시판 등에 회원이
    직접 등록한 게시물, 댓글 등은
    탈퇴 후에도 삭제되지 않습니다.
  • 탈퇴한 아이디는
    ‘회원 ID 영구보관’ 정책으로
    재가입 시에도 다시 사용할 수 없습니다.
    (다른회원도 사용불가)
